Better Lamps · wireframe sketches · v0.1

Redesign exploration —
two directions, dense & warm.

The current site reads as inventory software circa 2014: messy modal forms, decorative chart backgrounds with no actual charts, inconsistent type, no rhythm. These wireframes commit to a single material — warm paper and ink, with a single coral accent — and explore two ways to organise the workspace. Both are dark-mode-first when shipped; shown here in paper/ink so the structure reads cleanly.

Scope   9 screens × 2 directions Density   Tight Surface   Desktop ≥ 1280 Mode   Dark primary, light option
§ 01

The material

cream + ink + one coral

One serif, one sans, one handwritten note voice. One warm coral that does all the work — accents, primary actions, key totals, "this matters" markers. Everything else is paper, ink, or muted rule.

01

Coral pays rent

Coral marks one thing per surface — the primary action, the active row, the running total. Never decoration.

02

No fake charts

The old stat cards had chart-shaped wallpaper. Numbers stand alone. If a chart isn't real, it isn't there.

03

One row, one rhythm

Every table uses the same column gap, row height, thumbnail size, money treatment. Same shape across screens.

04

Forms aren't pop-ups

Slide-over modals covering the table are the biggest pain point. Both directions kill them — different ways.

Direction A

The Workshop.

A tool you sit inside all day. Compact left rail you never lose. Forms expand inline where the data lives — no slide-over covering half the screen. Dense, quiet, fast.

signature moves
  • Persistent rail nav with active-coral marker
  • Global ⌘K search bar in the topbar
  • Inline expanding forms inside tables
  • Stat strip — numbers only, no chart wallpaper
  • Status pill, stock bar, money pattern fixed once
